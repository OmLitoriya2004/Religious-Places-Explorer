# Religious Places Exploration Platform

The Religious Places Exploration Platform is a web application designed to explore and document various religious sites across India. It allows users to add, view, edit, and delete information about temples, gurudwaras, churches, and masjids.

## Features

- **Explore Religious Places**: Users can browse through a collection of religious sites.
- **Add New Places**: Users can contribute by adding new religious places to the platform.
- **Edit Existing Places**: Users can update information about existing places.
- **View Detailed Information**: Each place has a detailed page with information including name, description, entry fee, nearby hotels, and location.

- ## Technologies Used
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: Node.js, Express.js, REST
Database: MongoDB
Templating Engine: EJS (Embedded JavaScript)

Religious Places Exploration Platform
The Religious Places Exploration Platform is a web application designed to explore and document various religious sites across India. It allows users to add, view, edit, and delete information about temples, gurudwaras, churches, and masjids.

Features
Add New Places: Users can add new religious places with details such as title, description, entry fee, location, nearby hotels, and an image URL.

View Places: Browse through a list of existing religious places, each displaying essential information.

Edit Places: Modify existing place details, including updating descriptions, entry fees, nearby hotels, and images.

Delete Places: Remove places from the platform when no longer needed.


- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (with Mongoose ORM)
- **Templating Engine**: EJS (Embedded JavaScript)


## Usage

- **Adding a New Place**:
- Click on the "Add New Place" in Navbar.
- Fill out the form with details about the new place.
- Click "Add" to submit the form.

- **Viewing and Editing Places**:
- Click on a place from the list to view its details.
- Click "Edit" to modify the place information.
- Update the details and click "Edit" again to save changes.

- **Deleting a Place**:
- Click "Delete" on the place details page to remove the place from the platform.
